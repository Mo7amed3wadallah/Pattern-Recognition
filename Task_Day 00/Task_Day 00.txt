>> - The mean: is the average in a set of data.

       It would only be used on quantitative data.

- The median: is the middle number in a set of data, when all values which comprise a given dataset have been ordered from the lowest value to the highest value.

- The mode: is the most frequently reported value in a set of data. It is not a commonly reported measure of central tendency.

- The variance: is an unstandardized measure of spread. 
  While it is often reported in statistical literature, the standard deviation will be more commonly encountered as it is a standardized measure of spread. 
 The standard deviation represents the average of the sum of squared deviations from the mean.
